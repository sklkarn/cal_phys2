If there can be even number of segments of \(360^{o}\) then the number of images formed is given by n-1 or if the number of segments of \(360^{o}\) is odd then number of images formed will be n only. That is if \(\theta = 90^{o}\) then number of images,
\begin{equation*} n=\frac{360^{o}}{90}-1=3 \end{equation*}
and if \(\theta = 72^{o} \text{,}\) then number of images,
\begin{equation*} n=\frac{360^{o}}{72}=5\text{.} \end{equation*}
