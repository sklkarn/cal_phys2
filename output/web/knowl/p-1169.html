But this is only apparent path difference, from electromagnetic theory of light it is known that when light wave reflected from the surface of optical denser medium, it suffers a phase difference of \(\pi\text{,}\) which is equivalent to a path difference of \(\frac{\lambda}{2}\text{,}\) i.e., a wave along AP has already a path difference of \(\frac{\lambda}{2} \text{.}\) Hence, the actual path difference in this case is
\begin{equation*} \Delta x= 2t\eta_{2}\cos r - \frac{\lambda}{2} \end{equation*}
  1. Now if \(\Delta x = n\lambda\text{,}\) where \(n = 0, 1, 2, 3,\cdots,\) then constructive interference takes place and the film appears bright, i.e.,
    \begin{equation*} 2t\eta_{2}\cos r - \frac{\lambda}{2} = n\lambda \end{equation*}
    \begin{equation} 2\eta t \cos r =(2n+1)\frac{\lambda}{2} \tag{5.8.6} \end{equation}
  2. If \(\Delta x = (2n+1)\frac{\lambda}{2}\text{,}\) where \(n=0,1,2,3,\cdots,\) then destructive interference takes place and the film appears dark, i.e.,
    \begin{equation*} 2t\eta_{2}\cos r - \frac{\lambda}{2} = (2n+1)\frac{\lambda}{2} \end{equation*}
    \begin{equation*} 2\eta t \cos r =(2n+1)\frac{\lambda}{2} +\frac{\lambda}{2}= (n+1)\lambda \end{equation*}
    \begin{equation} 2\eta t \cos r =n\lambda\tag{5.8.7} \end{equation}
    \([\because (n+1)=n,\quad \text{for}\quad n=0,1,2,\cdots] \)
Since the amplitude is varying for the interfering waves, fringes are not perfect.
