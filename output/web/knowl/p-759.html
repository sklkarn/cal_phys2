Sound Wave Equations are
\begin{equation*} s(x,t) = s_{o} sin (kx-\omega t) \quad (Displacement) \end{equation*}
\begin{equation*} p(x,t) = Bks_{o} cos (kx-\omega t) \quad (Pressure) \end{equation*}
\begin{equation*} p_{max} = |Bks_{o}| \quad \text{(Pressure Amplitude)} \end{equation*}
Now sound intensity can be
\begin{equation*} I = \frac{P}{A} (W/m^{2}) = \frac{\,E}{A\,dt}=\frac{F \,ds}{A\,dt} = p(x,t)v(x,t)=Bk\omega s_{o}^{2}\cos^{2}(\omega t-kx) \end{equation*}
At any point x the average value of function \(\cos^{2}(\omega t-kx) \) over one period \(T=2\pi/\omega\) is \(\frac{1}{2}\text{.}\)
\begin{equation*} \therefore\quad I = \frac{1}{2}Bk\omega s_{o}^{2} = \frac{\omega p_{max}^{2}}{2Bk}=\frac{v p_{max}^{2}}{2B} \end{equation*}
\begin{equation*} \text{But,}\quad v=\sqrt{\frac{B}{\rho}}\quad \text{hence}\quad I = \frac{1}{2}\sqrt{\rho B}\omega^{2}s_{o}^{2} \end{equation*}
\begin{equation*} \text{or,}\quad I = \frac{p^{2}_{max}}{2\rho v} = \frac{p^{2}_{max}}{2\sqrt{\rho B}} \end{equation*}
