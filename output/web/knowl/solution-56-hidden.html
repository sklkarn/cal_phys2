Magnetic field at a distance, x from current carrying conductor is given by
\begin{equation*} B= \frac{\mu_{o}I}{2\pi x}. \end{equation*}
Now magnetic flux through a differential area \(\,dA \) is given by
\begin{equation*} \,d\Phi=\vec{B}\cdot\vec{\,dA} = B\,dA = BL\,dx \end{equation*}
\begin{equation*} \therefore\quad \Phi = \int\,d\Phi= \int\limits_{h}^{h+w} \frac{\mu_{o}IL\,dx}{2\pi x} = \left.\frac{\mu_{o}IL}{2\pi} \ln x\right]_{h}^{h+w} \end{equation*}
\begin{equation*} = \frac{\mu_{o}IL}{2\pi} ln\left(\frac{h+w}{h}\right) = \frac{\mu_{o}L}{2\pi} (a+bt) ln\left(\frac{h+w}{h}\right) \end{equation*}
Magnetic field is in decreasing order as point P shifts away form the current carrying conductor but flux increases on every layer in a loop with time as current increases in a wire.
\begin{equation*} \therefore\quad \varepsilon = - \frac{\,d\Phi}{\,dt} = - \frac{\mu_{o}L}{2\pi}ln\left(\frac{h+w}{h}\right)b \end{equation*}
As the current changes in a wire, it changes the magnetic flux into the loop. Hence from Lenzâ€™s law the direction of induced current in a loop must opposes this change, meaning the direction of current must be counterclockwise if current increases in a wire.
