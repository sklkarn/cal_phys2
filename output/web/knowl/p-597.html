We can write
\begin{equation*} I = \omega C E_{o}e^{j\omega t}e^{j\frac{\pi}{2}} = \omega C E_{o}e^{j(\omega t+\frac{\pi}{2})} \end{equation*}
\begin{equation*} = \frac{E_{o}}{1/\omega C}e^{j(\omega t+\frac{\pi}{2})}= I_{o}e^{j(\omega t+\frac{\pi}{2})} \end{equation*}
where,
\begin{equation*} I_{o}= \frac{E_{o}}{1/\omega C} \end{equation*}
is the peak value of altenating current,
\begin{equation*} I=I_{o}\sin\left(\omega t+\frac{\pi}{2}\right) \end{equation*}
Hence, in this case also current is alternating sinusoidal with frequency \(\omega\) but leading the emf by \(\frac{\pi}{2}\) phase as shown in FigureÂ 3.9.5.(b).
