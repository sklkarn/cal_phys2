Consider a current carrying straight conductor of length 2a. To find a magnetic field at a point P from the conductor consider a small current element \(\,dy\) on a conductor and use Biot-Savartâ€™s law,
\begin{equation*} \vec{\,dB}=\frac{\mu_{o}I}{4\pi}\frac{\vec{\,dy}\times\hat{r}}{r^{2}} (-\hat{k}) \end{equation*}
The direction of magnetic field at P is given by right hand grip rule which is going into the plane of the paper. \(\hat{r} \) is unit vector along \(\vec{r}\) and \(\theta\) is angle between current element and line joining the current element to point P.
\begin{equation*} \text{or,}\quad \,dB=\frac{\mu_{o}I}{4\pi}\frac{\,dy\sin\theta}{r^{2}} =\frac{\mu_{o}I}{4\pi}\frac{x}{r}\frac{\,dy}{r^{2}} \end{equation*}
\begin{equation*} = \frac{\mu_{o}I}{4\pi}\frac{x\,dy}{r^{3}} =\frac{\mu_{o}I}{4\pi}\frac{x\,dy}{(x^{2}+y^{2})^{3/2}} \end{equation*}
\([\because \quad r^{2}=x^{2}+y^{2}]\)
\begin{equation} \therefore \quad B = \int \,dB=\frac{\mu_{o}I x}{4\pi}\int\limits_{-a}^{a}\frac{\,dy}{(x^{2}+y^{2})^{3/2}} \tag{3.4.1} \end{equation}
Now solve the integrand first with the help of geometry of the figure then substitute the value to obtain B.
\begin{equation*} \tan\phi = \frac{y}{x} \end{equation*}
\begin{equation*} \text{or,}\quad y=x\tan\phi \end{equation*}
\begin{equation*} \text{or,}\quad \,dy=x\sec^{2}\phi\,d\phi \end{equation*}
\begin{equation*} \therefore \quad \mathscr{I} = \int \frac{\,dy}{(x^{2}+y^{2})^{3/2}} = \int \frac{x \sec^{2}\phi\,d\phi}{\left[x^{2}(1+\tan^{2}\phi)\right]^{3/2}} \end{equation*}
\begin{equation*} = \int\frac{x\sec^{2}\phi\,d\phi}{x^{3}\sec^{3}\phi} = \frac{1}{x^{2}}\int\cos\phi\,d\phi \end{equation*}
\begin{equation*} \text{or,}\quad \mathscr{I} = \frac{1}{x^{2}}\sin\phi =\frac{1}{x^{2}}\frac{y}{r} =\frac{y}{x^{2}\sqrt{x^{2}+y^{2}}} \end{equation*}
