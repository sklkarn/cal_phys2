Item c.

\begin{align*} V \amp = -\int\vec{E}\cdot\vec{\,dl}\\ \amp =-\left(\int E_{x}\,dx+\int E_{y}\,dy\right) \\ \amp = -c\left(\int y^{2}\,dx+\int 2xy\,dy\right) \end{align*}
This must be true along any path, so choose the path \(y = 0, \quad \,dy = 0,\) while x goes from 0 to x; then hold x constant with \(\,dx = 0\) while y goes from 0 to y. For the first part of this path, \(V = 0,\) and for the second part
\begin{equation*} V(x,y) = -c\int\limits_{0}^{y} 2xy\,dy = -c x \left[y^{2}\right]_{0}^{y}= -c x y^2 \end{equation*}
