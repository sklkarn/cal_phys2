For path BC: \(x = 1 m, \quad \,dx = 0,\) and y goes from 0 to 2 m, so
\begin{equation*} W_{BC} = xqc\int\limits_{0}^{2} 2y \,dy = xqc \left[y^{2}\right]_{0}^{2} = 8.00\times 10^{-3}\,J \end{equation*}
For total path
\begin{equation*} W_{ABC} = W_{AB} + W_{BC} = 8.00 \times 10^{-3} \,J \end{equation*}
