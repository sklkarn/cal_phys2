Toroid is considered as an infinitely long solenoid or an endless solenoid as shown in Figure 3.5.5. Since magnetic field outside a solenoid is zero we can have magnetic field only inside a toroid. To find magnetic field at any point inside the toroid consider an Amperean loop of radius r so that it encloses the current which is going into the page then from Ampere’s law
\begin{equation*} \oint\vec{B}\cdot\,dl= B_{in}\left.\,dl\right\vert_{0}^{2\pi r} = B(2\pi r) =\mu_{o}I_{enc} =\mu_{o}n(2\pi r)I \end{equation*}
\begin{equation*} \therefore\quad B_{in}= \mu_{o} n I \end{equation*}
where n is number of turns per unit length and I is current following in a toroid.
