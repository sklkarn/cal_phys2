Item 2.

When phase difference \(\delta = \pi, 3\pi, \cdots, (2n+1)\pi,\) or the path difference \(x=\frac{\lambda}{2}, 3\frac{\lambda}{2}, 5\frac{\lambda}{2}, \cdots, (2n+1)\frac{\lambda}{2} \text{.}\)
\begin{equation*} I=0 \end{equation*}
Intensity is minimum when the phase difference between the interfering waves is an odd number multiple of \(\pi\) or the path difference between them is an odd number multiple of \(\frac{\lambda}{2}\text{.}\)
