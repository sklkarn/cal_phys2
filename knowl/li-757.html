Item 4.

Interference:
\begin{equation*} \Delta x = n \lambda, \quad n = 0, 1, 2,\cdots \text{(Constructive Interference)} \end{equation*}
\begin{equation*} \Delta x = \frac{n}{2} \lambda, \quad n = 1, 3, 5,\cdots\text{(Destructive Interference)} \end{equation*}
\begin{equation*} \Delta \theta = 2\pi \frac{\Delta x}{\lambda}\quad \text{(Phase Angle)} \end{equation*}
