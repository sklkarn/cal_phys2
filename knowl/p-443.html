  1. \begin{equation*} \therefore \quad B = \frac{\mu_{o}I x}{4\pi}\left[\frac{y}{x^{2}\sqrt{x^{2}+y^{2}}}\right]_{-a}^{a} \end{equation*}
    \begin{equation*} =\frac{\mu_{o}I}{4\pi} \left[\frac{2a}{x\sqrt{x^{2}+a^{2}}}\right] \end{equation*}
  2. When \(x \lt \lt a\) (ie., for infinitely long wire) then \(1 \lt \lt \frac{a}{x} \text{,}\) hence
    \begin{equation*} B = \frac{\mu_{o}I}{4\pi} \left[\frac{2a}{x\cdot x\sqrt{1+\frac{a^{2}}{x^{2}}}}\right] \end{equation*}
    \begin{equation*} = \frac{\mu_{o}I}{4\pi}\frac{2a}{x^{2}\sqrt{\frac{a^{2}}{x^{2}}}} =\frac{\mu_{o}I}{4\pi} \frac{2a}{x} \end{equation*}
    \begin{equation*} \therefore \quad B= \frac{\mu_{o}I }{2\pi x} \end{equation*}
  3. When \(x \gt \gt a\) then \(\frac{x}{a} \gt \gt 1\text{,}\) hence
    \begin{equation*} B = \frac{\mu_{o}I }{2\pi} \frac{a}{x^{2}} \simeq 0 \end{equation*}
